# Codex 本地客户端配置

[Codex](https://openai.com/codex/) 是 OpenAI 的 AI 编码工具。本页聚焦会读取本地 Codex 配置的使用场景，例如 Codex CLI 和本地客户端。通过 Look2Eye，你可以获得高配额、低延迟，以及全球顶级模型的统一调用能力。

## 为什么使用 Look2Eye？

- **高速率配额** — 默认高 RPM，满足高频编码需求
- **低延迟** — 全球加速节点，自动最优路由
- **全球顶级模型** — OpenAI、Claude、Gemini 等主流模型统一接入
- **完整协议支持** — 100% 兼容 OpenAI 协议
- **WebSocket 模式** — 推荐开启 Responses 协议格式和 WebSocket 长连接，长时间编码会话更流畅
